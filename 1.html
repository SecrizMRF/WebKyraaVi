Mengenal HTML, CSS, JS dan Tailwind CSS

HTML

HTML (Hypertext Markup Language). Bayangkan kamu ingin membangun sebuah robot, maka HMTL ini adalah rangka dari robot tersebut. Pada umumnya, rangka itu terdiri dari kepala dan bagian badan. Begitu juga pada HTML, terdapat Head dan Body. Pada HTML juga lah kita dapat mendefinisikan teks, hyperlink, tautan, gambar, elemen grafis dan tautan.

FYI: HTML diperkenalkan oleh tim Berners-Lee pada tahun 1991

CSS

CSS (Cascading Style Sheet), namanya persis mendefinisikan fungsi dirinya. Tadi kita menganalogikan HTML sebagai rangka dari sebuah robot, sekarang kita bisa menganalogikann CSS sebagai skin atau tampilan dari si robot yang ingin kita ciptakan. CSS digunakan untuk memperindah tampilan dari HTML kita, mulai dari pengaturan sederhana, sampai pengaturan mendetail. Misalnya yang sederhana, warna teks, ukuran teks, warna background. CSS juga bisa mengatur posisi atau ukuran dari sebuah elemen HTML.

Js

JS - JavaScript

JS (JavaScript). Sekarang robot kita udah berbentuk, udah bisa berdiri, udah bisa bergerak, udah ganteng, cantik udah rapi. Tapi, kalau kita membangun sebuah robot, yang harus kita pikirkan bukan hanya tentang rangka dan penampilannya saja, tapi juga kecerdasannya. Jadi, kita perlukan "otak" untuk robot kita. Beginilah kita analogikan JavaScript. JavaScript adalah seperti otak di balik web kita. JavaScript juga lah yang memungkinkan pengguna dapat berinteraksi dengan web kita.

Tailwind CSS

Tailwind CSS. Bukan ekor angin ya seperti namanya. Tailwind ini adalah framework CSS yang dapat kita gunakan. Tujuannya, untuk empercepat dan mempersingkat proses pembuatan website kita dengan menyediakan kelas utilitas yang dapat kita kustomisasi langsung di dalam HTML, tanpa perlu file CSS yang terpisah.
