Langkah Membuat Website

Kamu

Halo! Aku mau mulai belajar bikin website sederhana nih. Langkah pertama yang harus aku lakukan apa ya?

Kyra

Langkah awal: buat folder project dulu di komputer kamu, misalnya dengan nama MyWebsiteKu. Di dalamnya, buat tiga file: index.html, style.css, dan script.js.

Kamu

Oke! Terus di dalam index.html aku harus isi apa?

Kyra

Di situ kamu tulis struktur dasar HTML. Misalnya:
<!DOCTYPE html>
<html>
<head><title>Website Sederhana</title></head>
<body><h1>Halo Dunia!</h1></body>
</html>

Kamu

Nicee. Sekarang gimana cara aku tambahin warna atau style-nya?

Kyra

Kamu buka file style.css dan tambahkan misalnya:
body { background-color: #0f172a; color: white; font-family: Poppins; }
Lalu hubungkan ke index.html pakai tag: <link rel="stylesheet" href="style.css">

Kamu

Kalau aku mau tambahin tombol yang bisa diklik pakai JavaScript gimana?

Kyra

Gampang! Di index.html tambahkan tombol:
<button id="klikAku">Klik Aku</button>
Lalu di script.js tulis:
document.getElementById
("klikAku").addEventListener
("click", () => alert("Tombol diklik!"));
